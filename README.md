# Apply to your own Prompt

1. Add API Key to the `apikey.txt` and the system prompt
3. Replace the data.csv with your own data.csv. It should have two columns: id and text
4. Open the api_coding.ipynb with the already included .venv; the .venv already contains all python libraries for the script
5. Change the `output_fields` variable in the jupyter notebook.
6. Run and pray
